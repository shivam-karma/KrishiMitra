# Firebase Phone OTP Setup for KrishiMitra

## ✅ What Was Implemented
- Firebase Phone Authentication replaces Fast2SMS
- Real SMS sent to any Indian number via Firebase
- Pinput OTP UI (6-box PIN field)
- Resend OTP with 60-second countdown
- Android auto-verification (OTP fills automatically)
- Flask `/auth/firebase-verify` syncs Firebase user to your SQLite DB

---

## 📋 Step-by-Step Setup

### 1. Create Firebase Project
1. Go to https://console.firebase.google.com
2. Click **Add project** → name it `krishimitra`
3. Disable Google Analytics (not needed) → **Create project**

### 2. Enable Phone Authentication
1. In Firebase Console → **Build** → **Authentication**
2. Click **Get started** → **Sign-in method** tab
3. Click **Phone** → toggle Enable → **Save**

### 3. Add Test Numbers (Free — No SMS cost)
1. Still in Phone provider → **Phone numbers for testing**
2. Add your number: `+91XXXXXXXXXX` with code `123456`
3. Add up to 10 test numbers — these work **without billing**

### 4. Add Android App to Firebase
1. Click the **Android** icon on Firebase project overview
2. **Package name**: `com.example.krishimitra_app`
   *(check `android/app/build.gradle` → `applicationId`)*
3. Download `google-services.json`
4. Place it at: `krishimitra_app/android/app/google-services.json`

### 5. Update Android build files

**`android/build.gradle`** — add inside `buildscript > dependencies`:
```groovy
classpath 'com.google.gms:google-services:4.4.2'
```

**`android/app/build.gradle`** — add at the very bottom:
```groovy
apply plugin: 'com.google.gms.google-services'
```

### 6. Initialize Firebase in Flutter main.dart

Edit `lib/main.dart` — add these lines:

```dart
import 'package:firebase_core/firebase_core.dart';
import 'firebase_options.dart'; // generated by FlutterFire CLI

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await Firebase.initializeApp(
    options: DefaultFirebaseOptions.currentPlatform,
  );
  runApp(const ProviderScope(child: KrishiMitraApp()));
}
```

### 7. Generate firebase_options.dart (recommended)
```bash
dart pub global activate flutterfire_cli
flutterfire configure --project=YOUR_FIREBASE_PROJECT_ID
```
This auto-generates `lib/firebase_options.dart`.

## 🏗️ Architecture (After Setup)

```
Flutter App
  └─ Firebase Phone Auth SDK
       ├─ Sends SMS (Firebase manages it)
       ├─ User enters OTP in Pinput widget
       ├─ Firebase verifies → returns credential
       └─ App calls POST /auth/firebase-verify → Flask
            └─ Creates/fetches User in SQLite
```
